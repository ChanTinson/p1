Class SocketAPI

webSocket的基础封装 本类定位于解决H5及Native中webSocket的各种问题，由于是基础封装，所以本类仅包含以下功能 1、与服务器的链接、断开、异常捕获和接收及发送数据 2、链接状态维护 注意：本类不包括心跳保活功能，实际使用过程中如有需要请自行扩展。
