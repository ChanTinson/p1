Class TimerManager

定时器管理器 本类为setTimeout及setInterval和Timer的替代实现，主要是方便程序中Timer事件的生命周期管理，使用起来更为便利。 程序中如用到setTimeout，setInterval及Timer的地方应尽量选择此方案。
